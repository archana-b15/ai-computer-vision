# AI Computer Vision Projects 👁️

A collection of Computer Vision projects built 
with HuggingFace Transformers and Python.

## Projects

### 1. Object Detector
Detects and localizes multiple objects in any image.
- Model: `facebook/detr-resnet-50`
- Input: Any image URL
- Output: Image with bounding boxes and confidence scores
- Can detect 91 different object categories

## Technologies Used
- Python
- HuggingFace Transformers
- PIL (Pillow)
- Matplotlib
- Google Colab


### 2. Face Emotion Detector
Detects human emotions from any face photo.
- Model: `trpakov/vit-face-expression`
- Input: Any face image URL
- Output: Emotion label + confidence score
- Detects 7 emotions: happy, sad, angry, 
  surprised, fearful, disgusted, neutral

### 3. Image Caption Generator
Generates natural language descriptions for any image.
- Model: `Salesforce/blip-image-captioning-base`
- Input: Any image URL
- Output: Natural language caption
- Combines Computer Vision + NLP (Multimodal AI)
